Getting started
Components

Install

flame-ui based on styled-components and styled-system so you don't need to install them.

Package

To install the main package, run

yarn add @flame-ui/react

Fonts

the flame-ui library uses Open Sans as a common font. We assume that you will use your custom fonts, but if you want to use the default font, be sure to include it in the head

<link
  href="https://fonts.googleapis.com/css2?family=Open+Sans:wght@400;700&display=swap"
  rel="stylesheet"
/>

Icons

flame-ui icons are available as a separate package, to install it run

yarn add @flame-ui/icons
